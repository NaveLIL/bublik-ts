# Bublik Bot

Дискорд-бот для EREZ. TypeScript, discord.js, PostgreSQL, Redis. Всё в докере.

## Что умеет

- **Экономика** — шекели, ежедневки, работа, крим, магазин ролей, казино (блэкджек, монетка, кости, слоты), ограбления, лидерборд
- **Полковые бои** — отряды, панель управления, пинг рекрутов, приказы, выговоры с апелляциями
- **Приветствия** — эмбед при входе/выходе, кнопки с правилами, авто-роль, напоминания
- **Временные войсы** — генераторы каналов, полная панель (лок, хайд, лимит, кик, трансфер и т.д.), награды за время
- **Отпуска** — заявки, модерация, дм-уведомления, автоотклонение просрочки
- **Утилиты** — `/ping`, `/reload` (хот-релоад модулей), `/language` (смена языка), `/info`
- **Локализация** — русский + английский, переключается посерверно

## Запуск

```bash
cp .env.example .env
# вписать DISCORD_TOKEN и DISCORD_CLIENT_ID

docker compose up -d --build
```

Бот сам накатит схему БД при старте.

Если хочешь локально без докера:
```bash
npm install
npx prisma generate
npx prisma db push
npm run dev
```

## Команды

`/info`, `/ping` — глобальные, остальные гильдовые:

`/balance`, `/daily`, `/weekly`, `/work`, `/crime`, `/beg`, `/rob`, `/shop`, `/coinflip`, `/dice`, `/slots`, `/blackjack`, `/leaderboard` — экономика

`/economy` — настройка экономики (админ)
`/regbattle` — полковые бои (админ)
`/welcome` — приветствия (админ)
`/voice` — временные войсы (админ)
`/vacation` — отпуска
`/language` — язык бота на сервере (админ)
`/reload` — перезагрузка модуля без рестарта (админ)

## Стек

Node.js 20, TypeScript, discord.js v14, PostgreSQL 16 + Prisma, Redis 7, Winston, Docker Compose

## Сборка

```bash
npm run build:protected    # ts -> js -> обфускация
```

В репо лежит `dist-protected/` — обфусцированная сборка, исходники не включены.
